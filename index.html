🎯 今日脑洞题

一家工厂每月生产1000个产品，安全库存设为200个，现有库存300个，在途100个。如果下月计划生产1000个，净需求是多少？
净需求 = 毛需求 - 现有库存 - 在途 + 安全库存缺口
可用库存 = 300 + 100 = 400个
净需求 = 1000 - 400 = 600个（可用库存>安全库存，无需补充安全库存）
✅ 答案：净需求 = 600个
